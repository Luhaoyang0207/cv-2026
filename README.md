# Haoyang Lu - Resume

This is a modern, responsive HTML/CSS resume created from `Haoyang Lu (CV).pdf`.

## Features
- **Responsive Design**: Works on mobile and desktop.
- **Glassmorphism Theme**: Premium dark mode aesthetic.
- **Interactive**: Hover effects and smooth scrolling.
- **Downloadable PDF**: Includes a direct link to the original PDF.

## How to Deploy to GitHub Pages

1. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```

2. **Create a Repository on GitHub**:
   - Go to GitHub and create a new repository (e.g., `resume`).

3. **Push to GitHub**:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/resume.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**:
   - Go to your repository **Settings** > **Pages**.
   - Select the `main` branch as the source and click **Save**.
   - Your resume will be live at `https://YOUR_USERNAME.github.io/resume/`.

## Local Development
To view the resume locally, simply open `index.html` in your browser.
